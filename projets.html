Projet 3 : Activité Pratique JavaScript : Ecrire des scripts Javascript et les intégrer dans HTML

L'objectif de cette activité était de maîtriser les fondamentaux du JavaScript en pratiquant l'intégration de scripts dans des fichiers HTML, notamment à travers l'usage des instructions conditionnelles et des structures de boucle (for, while, do-while). J'ai ainsi appris à gérer les interactions avec l'utilisateur via prompt() et à appliquer des algorithmes de base pour le calcul et la génération de séquences. J'ai surtout retenu l'importance d'isoler mon code dans des fonctions et d'utiliser la méthode fiable document.getElementById().innerHTML pour l'affichage, résolvant ainsi les problèmes d'exécution dans les navigateurs modernes.

Technologies utilisées : HTML, Javascript.
